## Install

### Mac/Linux

```bash
git clone https://github.com/m1chaelwilliams/hacksu2025.git --branch ai
cd hacksu2025
python3 venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt
python3 -m main
```

### Windows

```bash
git clone https://github.com/m1chaelwilliams/hacksu2025.git --branch ai
cd hacksu2025
python venv .venv
.venv/Scripts/Activate.ps1
pip install -r requirements.txt
python -m main
```
